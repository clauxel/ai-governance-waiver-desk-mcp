# Workflow

AI Governance Waiver Desk is a paid remote MCP approval gate for AI use cases, model risk, exception reasons, owners, approvers, expiry dates, and governance receipts.

## Repeatable Flow

1. Submit use case, model, risk level, exception reason, owner, and expiry date.
2. Evaluate policy fit, risk class, and required approval path.
3. Return allow, review, deny, or expired with structured reasons.
4. Archive a waiver receipt and export governance evidence.

## Output Mindset

The useful artifact is not a marketing claim. It is evidence that a reviewer can inspect, export, and compare later.
